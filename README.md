# Together Mod Charts Manager (TMCM)

> ADOFAI / Together Mod 谱面管理工具 – 导入、浏览、播放、编辑、在线查找、批量导出，一站式搞定。

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey)]()

**Together Mod Charts Manager（TMCM）** 是一款面向 ADOFAI / Together Mod 社区的桌面端谱面管理工具，基于 PyQt5 打造，深色 Catppuccin 风格界面。  
它可以一键导入 `.zip` 谱面包，自动解析 `.adofai` 元数据，提供便捷的浏览、搜索、音乐播放、信息编辑功能，并直连 TUF 数据库在线查找谱面。导出时自动将你的修改写回 `.adofai` 文件，让谱面管理变得前所未有的简单。

---

## 功能亮点

- 批量导入：选择包含 `.zip` 谱面包的文件夹，自动解压并读取所有关卡的歌曲、作者、艺术家、难度、BPM 等完整信息。
- 内置播放器：使用 `pygame` 引擎播放 `.ogg` 背景音乐，支持播放 / 暂停 / 停止、进度拖动及播放结束自动停止。
- 强大搜索：支持按关卡名、歌曲名称、作者、艺术家、标签等多列模糊筛选，结果实时高亮。
- 信息编辑：选中单个关卡后可直接修改歌曲名、关卡作者、艺术家、难度（1-10）和描述，表格与详情面板同步更新。
- TUF 在线查找：通过 TUF API 一键搜索当前谱面是否已被收录。自动将多位作者拆分为独立查询条件，确保精准匹配。多个结果时弹出选择对话框，显示 PGU 难度图标及谱面详情，点击即可在浏览器中打开 TUF 页面。
- 批量导出：将选中的关卡导出到指定文件夹，并自动应用你编辑过的信息（修改会安全地写回导出的 `.adofai` 文件）。
- 暗色主题界面：采用 Catppuccin 配色方案，长时间使用也不伤眼。

## 快速开始

### 环境要求
- Python 3.8 或更高版本
- pip 包管理器

### 安装依赖
```bash
pip install PyQt5 pygame
```

### 运行
```bash
python main.py
```
首次启动时，程序会在自身所在目录下创建 `adofai_viewer_cache` 文件夹作为临时解压路径，无需额外配置。

## 使用指南

### 1. 加载关卡
1. 点击「路径配置」中的 **浏览** 按钮，选择包含 `.zip` 谱面文件的文件夹。
2. 程序会自动解压并解析所有 `.adofai`，加载完成后表格将显示全部关卡信息。
3. 加载过程中可以取消，已加载的关卡会保留在列表中。

> **提示**：如果你使用 Together Mod，其下载的谱面缓存默认位于：  
> `%AppData%\Local\Together\CachedCharts`  

### 2. 浏览与搜索
- **浏览**：单击任意行，右侧「关卡详情」面板将展示该关卡的完整信息（歌曲名、作者、艺术家、BPM、难度、描述等）。
- **搜索**：在顶部搜索框输入关键词，可在右侧下拉菜单中选择搜索列（全部、关卡名、歌曲名称、关卡作者、艺术家、标签）。筛选结果即时显示。

### 3. 音乐播放
- 选中一个拥有 `.ogg` 音乐文件的关卡，底部的音乐播放栏将被激活。
- 点击 **播放** 按钮开始播放，可随时 **暂停** 或 **停止**。
- 拖动进度条可跳转到任意位置。
- 播放结束后自动恢复为“播放”状态。

### 4. 编辑谱面信息
- 在表格中 **单选** 一个关卡，点击底部 **「编辑信息」** 按钮。
- 在弹出的对话框中修改 **歌曲名称、关卡作者、艺术家、难度、描述**。
- 确认后，表格中对应行的数据会立即更新，关卡名前出现 `*` 标记以示已编辑。
- 右侧详情面板也会同步刷新。

### 5. TUF 在线查找
- **单选** 一个关卡，点击紫色的 **「TUF 查找」** 按钮。
- 程序会根据关卡的歌曲名、艺术家及拆分后的作者列表向 TUF API 发起查询。
- **未找到**：弹出提示并给出可能原因（如信息不全、未上传至 TUF）。
- **找到唯一结果**：询问是否在浏览器中打开 TUF 关卡详情页。
- **找到多个结果**：弹出选择窗口，每个结果一行，展示 PGU 难度图标、歌曲、艺术家、作者、BPM、方块数和难度名。点击右侧 **「跳转」** 按钮即可在浏览器中查看该谱面。

### 6. 批量导出
- 使用 `Ctrl` 或 `Shift` 多选需要导出的关卡。
- 点击底部 **「导出选中关卡」**，选择目标文件夹。
- 导出时会自动应用你的信息修改（直接写入目标文件夹中的 `.adofai` 文件），文件夹命名格式为 `歌曲名称-关卡作者-ZIP包名`。

## 项目结构

```
.
├── main.py                        # 程序入口
└── adofai_viewer/                 # 主包
    ├── __init__.py

    ├── load_thread.py             # 后台解压与解析线程

    ├── tuf_searcher.py            # TUF API 搜索、异步请求、多结果选择对话框
    └── main_window.py             # 主窗口 UI 及全部业务逻辑
```

## 技术细节

- **音频播放**：由于 Qt5 默认不支持 OGG 解码，采用 `pygame.mixer` 播放音乐，并在首次需要播放时延迟初始化，避免启动延迟。
- **TUF 搜索**：自动将形如 `Nephrolepis & 2seehyun` 的作者字段拆分为 `creator:Nephrolepis, creator:2seehyun` 两个独立条件，保证 API 查询精确匹配。
- **编辑导出**：导出时通过正则替换 `.adofai` 文件中对应字段的值（区分字符串和数字），安全可靠且不破坏原有结构。
- **跨平台兼容**：所有文件操作均使用 `os.path` 并注意大小写不敏感查找，确保在 Windows、macOS、Linux 上均可运行。
- **深色主题**：基于 Catppuccin 配色的完整 QSS 样式表，提供一致且舒适的视觉体验。

## 常见问题

**Q: 为什么音乐无法播放？**  
A: 请确保已正确安装 `pygame`，且所选的关卡中包含 `.ogg` 格式的音乐文件。若文件存在但仍无法加载，请检查文件路径是否包含特殊字符或权限问题。

**Q: TUF 查找提示“未找到”但谱面确实在 TUF 上？**  
A: 可能是本地关卡信息不完整（缺少艺术家或作者）。尝试使用「编辑信息」功能补全真实数据后再重新搜索。此外请确保网络连接正常。

**Q: 导出的文件夹名称出现乱码或下划线？**  
A: 为了兼容各操作系统，程序会自动将中文等特殊字符转换为下划线。如希望保留中文，可自行修改 `utils.py` 中的 `sanitize_filename` 函数。
