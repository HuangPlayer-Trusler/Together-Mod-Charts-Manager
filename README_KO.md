# Together Mod Charts Manager (TMCM)

[English](README_EN.md) | [中文](README.md)

> ADOFAI / Together Mod 차트 관리 도구 – 가져오기, 탐색, 재생, 편집, 온라인 조회, 일괄 내보내기를 한 번에.

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**Together Mod Charts Manager (TMCM)** 는 ADOFAI / Together Mod 커뮤니티를 위한 데스크톱 차트 관리 유틸리티로, PyQt5 기반의 Catppuccin 다크 테마를 적용하였습니다.
`.zip` 차트 팩을 일괄 가져와 `.adofai` 메타데이터를 자동으로 파싱하고, 편리한 탐색, 검색, 음악 재생, 정보 편집 기능을 제공합니다. 또한 TUF 데이터베이스에 직접 연결하여 온라인 차트 조회가 가능하며, 내보내기 시 편집한 내용이 `.adofai` 파일에 반영됩니다.

---

## 주요 기능

- 일괄 가져오기: `.zip` 차트 팩이 있는 폴더를 선택하면 자동으로 압축을 풀고 곡명, 제작자, 아티스트, 난이도, BPM 등 모든 레벨 정보를 읽어옵니다.
- 내장 플레이어: `pygame` 엔진으로 `.ogg` 배경 음악을 재생하며, 재생/일시정지/정지, 진행 바 탐색, 재생 종료 후 자동 정지를 지원합니다.
- 강력한 검색: 레벨명, 곡명, 제작자, 아티스트, 태그 등 여러 열을 기준으로 실시간 필터링 및 강조 표시가 가능합니다.
- 정보 편집: 단일 레벨을 선택하여 곡명, 차터, 아티스트, 난이도(1~10), 설명을 수정할 수 있으며, 테이블과 상세 패널이 즉시 갱신됩니다.
- TUF 온라인 조회: TUF API를 통해 버튼 하나로 현재 차트가 등록되어 있는지 검색합니다. 여러 제작자가 있는 경우 개별 검색 조건으로 자동 분리하여 정확하게 매칭합니다. 결과가 여러 개일 경우 PGU 난이도 아이콘과 레벨 정보를 보여주는 선택 대화상자가 나타나며, 브라우저에서 바로 열 수 있습니다.
- 일괄 내보내기: 선택한 레벨을 대상 폴더로 내보내며, 편집한 내용이 안전하게 `.adofai` 파일에 기록됩니다.
- 다크 테마: Catppuccin 색상 조합으로 장시간 사용에도 눈이 편안합니다.

## 빠른 시작

### 요구 사항
- Python 3.8 이상
- pip

### 의존성 설치
```bash
pip install PyQt5 pygame
```

### 실행
```bash
python main.py
```
최초 실행 시 프로그램이 위치한 디렉토리에 `adofai_viewer_cache` 폴더가 임시 압축 해제 경로로 생성됩니다. 추가 설정은 필요 없습니다.

## 사용 가이드

### 1. 차트 불러오기
1. 경로 설정 섹션에서 **찾아보기** 버튼을 클릭하고 `.zip` 차트 파일이 있는 폴더를 선택합니다.
2. 프로그램이 모든 `.adofai` 파일의 압축을 풀고 파싱하면, 테이블에 레벨 정보가 표시됩니다.
3. 로딩 중에 취소할 수 있으며, 이미 로드된 레벨은 목록에 남아 있습니다.

> **팁**: Together Mod를 사용 중이라면, 다운로드된 차트 캐시의 기본 경로는 다음과 같습니다:  
> `%AppData%\Local\Together\CachedCharts`

### 2. 탐색 및 검색
- **탐색**: 행을 클릭하면 오른쪽 레벨 상세 패널에 전체 정보(곡명, 제작자, 아티스트, BPM, 난이도, 설명 등)가 표시됩니다.
- **검색**: 상단 검색창에 키워드를 입력하고, 드롭다운에서 검색할 열(전체, 레벨명, 곡명, 제작자, 아티스트, 태그)을 선택합니다. 결과가 실시간으로 필터링됩니다.

### 3. 음악 재생
- `.ogg` 음악 파일이 있는 레벨을 선택하면 하단 재생 바가 활성화됩니다.
- **재생** 버튼을 누르면 시작되며, 언제든지 **일시정지** 또는 **정지**할 수 있습니다.
- 진행 바를 드래그하여 원하는 위치로 이동할 수 있습니다.
- 재생이 끝나면 자동으로 “재생” 상태로 복귀합니다.

### 4. 레벨 정보 편집
- 테이블에서 레벨을 **단일 선택**한 후 하단의 **정보 편집** 버튼을 클릭합니다.
- 대화상자에서 **곡명, 차터, 아티스트, 난이도, 설명**을 수정합니다.
- 확인하면 테이블 행이 즉시 업데이트되고, 편집된 레벨 이름 앞에 `*` 표시가 나타납니다.
- 오른쪽 상세 패널도 함께 갱신됩니다.

### 5. TUF 온라인 조회
- 레벨을 **단일 선택**하고 보라색 **TUF 조회** 버튼을 클릭합니다.
- 프로그램은 레벨의 곡명, 아티스트, 분리된 제작자 목록을 바탕으로 TUF API를 검색합니다.
- **찾지 못한 경우**: 정보 부족이나 TUF 미등록 등의 이유를 알리는 대화상자가 나타납니다.
- **단일 결과**: 브라우저에서 TUF 레벨 페이지를 열지 묻습니다.
- **여러 결과**: PGU 난이도 아이콘, 곡명, 아티스트, 제작자, BPM, 타일 수, 난이도 이름이 표시된 선택 창이 나타납니다. **이동** 버튼을 클릭하면 해당 레벨을 브라우저에서 열 수 있습니다.

### 6. 일괄 내보내기
- `Ctrl` 또는 `Shift` 키를 사용하여 여러 레벨을 선택합니다.
- **선택한 레벨 내보내기** 버튼을 클릭하고 대상 폴더를 선택합니다.
- 내보내기 시 편집한 사항이 자동으로 적용되어 대상 폴더의 `.adofai` 파일에 기록됩니다. 폴더 이름 형식은 `곡명-차터-ZIP명`입니다.

## 기술 상세

- **오디오 재생**: Qt5가 기본적으로 OGG를 지원하지 않으므로, `pygame.mixer`로 음악을 재생하고 첫 재생 시 지연 초기화하여 시작 지연을 방지합니다.
- **TUF 검색**: `Nephrolepis & 2seehyun`과 같은 제작자 문자열을 `creator:Nephrolepis, creator:2seehyun` 개별 조건으로 자동 분리하여 정확한 API 매칭을 보장합니다.
- **편집 내보내기**: `.adofai` 파일의 JSON 필드를 정규식으로 교체(문자열과 숫자 구분)하여 파일 구조를 손상시키지 않습니다.
- **크로스 플랫폼**: 모든 파일 작업에 `os.path`를 사용하고 대소문자 구분 없는 폴백을 적용하여 Windows, macOS, Linux에서 원활하게 실행됩니다.
- **다크 테마**: Catppuccin 팔레트 기반의 완전한 QSS 스타일시트로 일관되고 편안한 UI를 제공합니다.

## 자주 묻는 질문

**Q: 음악이 재생되지 않는 이유는 무엇인가요?**  
A: `pygame`이 설치되어 있고 선택한 레벨에 `.ogg` 음악 파일이 포함되어 있는지 확인하세요. 파일이 존재함에도 로드되지 않는다면 경로에 특수 문자가 있는지, 또는 권한 문제가 있는지 점검하세요.

**Q: TUF 조회에서 “찾을 수 없음”이라고 뜨는데, 실제로 TUF에 등록된 차트입니다.**  
A: 로컬 레벨 정보가 불완전할 수 있습니다(아티스트나 제작자 누락). **정보 편집** 기능을 사용하여 실제 데이터를 채운 후 다시 검색해 보세요. 또한 인터넷 연결 상태를 확인하세요.

**Q: 내보낸 폴더 이름이 깨지거나 밑줄로 표시됩니다.**  
A: 여러 운영체제와의 호환성을 위해 프로그램이 한글 등 특수 문자를 밑줄로 변환합니다. 원래 이름을 유지하려면 `utils.py`의 `sanitize_filename` 함수를 수정하면 됩니다.
